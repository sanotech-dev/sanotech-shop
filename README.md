# Sanotech Shop

فروشگاه آنلاین حرفه‌ای با **Next.js 14 + Tailwind + MongoDB**


### ویژگی‌های نهایی
- صفحه محصول با تصاویر و توضیحات
- سبد خرید با ذخیره در localStorage
- پرداخت تستی
- پنل ادمین
- SSR برای سئو عالی
- ریسپانسیو کامل
- Dark Mode + انیمیشن

### تکنولوژی‌ها
- Next.js 14 (App Router)
- Tailwind CSS
- MongoDB (بعداً)
- Vercel برای دپلوی
